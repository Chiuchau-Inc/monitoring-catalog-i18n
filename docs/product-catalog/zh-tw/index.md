# 久朝智能風機監測系統型錄-繁中


![fan 1.jpg](../../assets/images/25c28296-80d6-46b7-8b65-5b615eaa49ab.jpg)

久朝企業有限公司推出的「久朝智能風機監測系統」，是針對工業用風機設備所設計的一站式**智慧監控與健康預測解決方案**。本系統結合**高精度感測器**、**AI異常預測演算法**與**即時數據視覺化平台**，協助企業掌握設備狀態、提前預防潛在故障，並同步監控設備能源使用與碳排放，協助企業達成**智慧維護與節能減碳**雙重目標。

### **系統特色**

✅ **即時監控**：整合三軸加速度感測器、溫度感測器與電力監控電表，24小時不間斷掌握風機運行狀態。數據每秒更新，異常即刻回報。

✅ **AI健康預測**：內建智能分析引擎，依ISO 20816標準自動評分健康值，提前辨識不平衡、軸彎曲、設備鬆動等隱患。及時通知維護團隊，使維修更具計畫性與預防性。

✅ **即時警報通知**：異常事件第一時間推送至**Line群組**與**Email**，提醒管理者快速應變，降低突發停機風險。

✅ **碳排放監測**：搭配智慧電表，精準記錄風機電力消耗，轉換即時碳排數據，協助企業監控**設備碳足跡**，符合環保法規與ESG趨勢。

✅ **月報表自動生成**：每月自動彙整設備健康趨勢、異常事件與碳排記錄，產出標準化PDF報表，方便內部管理與稽核報告使用。

### **系統架構**

![模組outline.svg](../../assets/images/35988382-579b-45ba-ae9d-77d29178d0c3.svg)

整體架構包含以下模組：

- **監測主機**：高性能單板電腦（四核心ARM處理器、內建128GB儲存、UPS保護模組）

- **感測器**：雙側配置**三軸加速度感測器**，監測馬達側與風輪側振動狀況

- **電力監控**：高精度RJ45 CT輸入電表，記錄電壓、電流、功率與電能消耗

- **通訊模組**：支援Wi-Fi、LoRa、4G LTE，彈性符合現場通訊環境

- **顯示器**（選配）：10.1吋工業級觸控面板，現場直觀操作

- **狀態燈**（選配）：三色狀態燈即時顯示設備健康狀況

### **安裝與維護概述**

- 獨立電源供電（110V/220V），確保監測模組穩定

- 感測器安裝於馬達側與風輪側，並參考**VAH方位**標準化安裝

- 電表安裝依馬達電壓與馬力選擇適合**CT比流器**

- 感測器、電表、主機、顯示器出廠預接線，現場免設定

- 支援遠端韌體更新與設備診斷
