# 久朝智能风机监测系统型录


![fan 1.jpg](../../assets/images/25c28296-80d6-46b7-8b65-5b615eaa49ab.jpg)

久朝企业有限公司推出的「久朝智能风机监测系统」，是一款专为工业风机设备设计的一站式智慧监控与健康预测解决方案。系统结合高精度传感器、AI异常预测算法与实时数据可视化平台，帮助企业全面掌握设备运行状态，提前预防潜在故障，同时监测设备能耗与碳排放，助力企业实现智能维护与节能减排双重目标。

### **系统特色**

**实时监控**：整合三轴加速度传感器、温度传感器与电力监控电表，24小时不间断掌握风机运行状态。数据每秒更新，异常立即上报。

**AI健康预测**：内置智能分析引擎，依据ISO 20816标准自动评分健康值，提前识别不平衡、轴弯曲、设备松动等隐患，并及时通知维护团队，提升维护的计划性和预见性。

**实时警报通知**：异常事件第一时间推送至Line群组与电子邮件（Email），提醒管理者迅速应对，降低突发停机风险。

**碳排放监测**：搭配智能电表，精准记录风机电力消耗，并实时转换碳排数据，帮助企业掌握设备碳足迹，符合环保法规与ESG趋势。

**月报表自动生成**：每月自动汇整设备健康趋势、异常事件与碳排放记录，生成标准化PDF报告，方便内部管理与审计使用。

### **系统架构**

![模組outline.svg](../../assets/images/35988382-579b-45ba-ae9d-77d29178d0c3.svg)

整体架构包含以下模块：

•**测主机**：高性能单板计算机（四核ARM处理器、内建128GB存储、UPS保护模块）

•**传感器**：双侧配置三轴加速度传感器，监测马达侧与风轮侧振动状态

•**电力监控**：高精度RJ45 CT输入电表，记录电压、电流、功率与能耗

•**通讯模块**：支持Wi-Fi、LoRa、4G LTE，灵活适应现场通讯环境

•**显示器（选配）**：10.1英寸工业级触控屏，方便现场操作

•**状态灯（选配）**：三色状态灯，实时显示设备健康状况
